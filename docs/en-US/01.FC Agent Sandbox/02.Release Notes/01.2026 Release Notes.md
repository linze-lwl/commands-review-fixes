# 2026 Release Notes

| **Feature** | **Change Type** | **Description** | **Related Documentation** |

| FC Agent Sandbox | General Availability | FC Agent Sandbox is generally available in the China North 2 (Beijing) region. It provides isolated cloud execution environments that you can create, run, connect to, and release on demand for AI agent workloads, code execution, command execution, file processing, and temporary service preview scenarios. Developers can connect to FC Agent Sandbox by using the E2B SDK or E2B CLI, and use core capabilities such as Sandbox, Commands, Filesystem, Code Interpreter, and Template. | [FC Agent Sandbox](../01.Getting%20Started/01.Product%20Overview.md) |
