# Manage Templates

Template management includes viewing templates, creating templates, tracking build status, and organizing template versions with tags and aliases.

## List templates

Use the E2B CLI:

```bash
e2b template list
```

You can also query templates through the E2B SDK. For the current API boundary, see [E2B Compatibility](../../01.Getting%20Started/05.E2B%20Compatibility.md).

## Create a template

Use `Template.build()` to create a custom template. For details, see [Build Custom Image Templates](03.Build%20Custom%20Image%20Templates.md).

## Build status and logs

When you build a template, keep the build ID and template ID so you can troubleshoot build failures, image pull failures, and dependency installation failures.

## Tags and aliases

Tags and aliases are useful for distinguishing template versions, business scenarios, and rollout stages. In production, create a new template, validate it, and then switch the template alias used by the application.
