# Built-in Templates

Built-in templates let you create a sandbox quickly. The quickstart and Code Interpreter examples use `code-interpreter-v1` by default.

## View templates

Use the CLI to view the templates available to the current account:

```bash
export E2B_API_KEY="<your-api-key>"
export E2B_API_URL="https://api.<region>.e2b.fc.aliyuncs.com"
export E2B_DOMAIN="<region>.e2b.fc.aliyuncs.com"

e2b template list
```

## Create a sandbox from a template

Python example:

```python
import os

sandbox = Sandbox.create(
    template="code-interpreter-v1",
    api_key=os.environ["E2B_API_KEY"],
    api_url=os.environ["E2B_API_URL"],
    domain=os.environ["E2B_DOMAIN"],
)
```

TypeScript example:

```typescript
const sandbox = await Sandbox.create("code-interpreter-v1", {
  apiKey: process.env.E2B_API_KEY,
  apiUrl: process.env.E2B_API_URL,
  domain: process.env.E2B_DOMAIN,
});
```
