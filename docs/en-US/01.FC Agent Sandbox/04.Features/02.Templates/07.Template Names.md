# Template Names

Template names are used to reference runtime environments when creating sandboxes. Good naming reduces accidental template misuse and version confusion.

## Naming recommendations

Include the business scenario, language or base image, key dependency versions, and a date:

```text
agent-python313-20260704
code-review-node22-20260704
data-analysis-py313-v1
```

## Use a template name when creating a sandbox

```typescript
const sandbox = await Sandbox.create("agent-python313-20260704", {
  apiKey: process.env.E2B_API_KEY,
  apiUrl: process.env.E2B_API_URL,
  domain: process.env.E2B_DOMAIN,
});
```

## Recommendations

Do not overwrite templates that are already in production use. Create a new template, validate it, and then switch application references.
