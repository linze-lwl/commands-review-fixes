# Base Images

Base images are the starting point for custom templates. FC Agent Sandbox currently recommends ACR EE images that are reachable over VPC as the build source.

## Image address format

```text
<acr-ee-vpc-domain>/<namespace>/<image>:<tag>
```

## Build a custom template

```python
import os
import time

from e2b import Template

connection = {
    "api_key": os.environ["E2B_API_KEY"],
    "api_url": os.environ["E2B_API_URL"],
    "domain": os.environ["E2B_DOMAIN"],
}

build = Template.build(
    Template().from_image(os.environ["FROM_IMAGE"]),
    name=f"agent-runtime-{int(time.time())}",
    **connection,
)

print(build.template_id)
```

## Recommendations

Keep the base image stable and traceable. In production, use an explicit version or digest instead of a drifting `latest`.
