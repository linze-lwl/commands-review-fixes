# Git Integration

FC Agent Sandbox can run Git commands inside the Sandbox, and can also use the Git module exposed by the E2B SDK for common repository operations. This is suitable when an agent needs to pull code, create temporary branches, run tests, or generate patches.

## Run Git with Commands

```typescript
const result = await sandbox.commands.run(
  "git clone --depth=1 https://github.com/e2b-dev/e2b.git /tmp/repo && git -C /tmp/repo status --short",
  { timeoutMs: 120_000 },
);

console.log(result.stdout);
```

## Use the SDK Git Module

```typescript
await sandbox.git.clone("https://github.com/e2b-dev/e2b.git", "/tmp/repo");
const status = await sandbox.git.status("/tmp/repo");
console.log(status);
```

## Recommendations

- When accessing private repositories, inject credentials from your application and enforce least privilege.
- For large repositories, prefer shallow clones or fetch only the required directories.
- Before running code from an untrusted repository, validate dependency sources, scripts, and execution commands first.
