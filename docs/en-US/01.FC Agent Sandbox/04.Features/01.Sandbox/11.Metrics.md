# Metrics

Sandbox Metrics lets you inspect runtime metrics such as CPU, memory, and disk usage. FC Agent Sandbox supports sandbox metrics and currently returns data at 1-minute granularity.

SDK example:

```typescript
const metrics = await sandbox.getMetrics();
console.log(metrics);

const metricsById = await Sandbox.getMetrics(sandbox.sandboxId, {
  apiKey: process.env.E2B_API_KEY,
  apiUrl: process.env.E2B_API_URL,
  domain: process.env.E2B_DOMAIN,
});
console.log(metricsById);
```

CLI example:

```bash
e2b sandbox metrics <sandbox-id>
```

The response includes the sampling time, CPU usage and core count, memory used and total, and disk used and total.

This capability is suitable for debugging, resource trend observation, and task troubleshooting. Use the Function Compute console, CloudMonitor, or Simple Log Service as the source of truth for billing, alerting, autoscaling, and capacity governance.
