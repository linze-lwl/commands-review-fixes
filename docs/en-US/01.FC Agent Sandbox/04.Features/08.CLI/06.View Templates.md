# View Templates

FC Agent Sandbox extends part of the template management workflow. Use the CLI to list the templates available to the current account:

```bash
e2b template list
```

The official E2B CLI documentation mainly expands sandbox-related commands and does not keep a dedicated `template list` page. When you work with FC Agent Sandbox, use the local CLI help and the FC Agent Sandbox compatibility guidance as the source of truth:

```bash
e2b template --help
e2b template list --help
```

The template list is useful for confirming template names, template IDs, and build results. For production sandbox creation, use templates that have already passed smoke testing.
