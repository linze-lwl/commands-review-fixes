# View Metrics

Use the CLI to inspect resource metrics for a running FC Agent Sandbox instance:

```bash
e2b sandbox metrics <sandbox-id>
```

The output includes CPU, memory, and disk metrics. For example:

```text
Metrics for sandbox <sandbox-id>:

[2026-07-05 13:48:00Z]  CPU: 1.895% /  2 Cores | Memory:    47 / 2048  MiB | Disk:     0 / 0     MiB
[2026-07-05 13:50:00Z]  CPU:   0.5% /  2 Cores | Memory:    50 / 2048  MiB | Disk:     0 / 0     MiB
```

Metrics currently return at 1-minute granularity. They are suitable for debugging, resource trend observation, and task troubleshooting. Use the Function Compute console, CloudMonitor, or Simple Log Service as the source of truth for billing, alerting, and capacity governance.
