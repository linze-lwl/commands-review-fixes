# Get Help

If you run into issues while using FC Agent Sandbox, it is recommended to verify the minimum example first, collect the required troubleshooting information, and then seek support through the Alibaba Cloud console, support tickets, or customer groups.

## Check before submitting an issue

- Confirm that you have completed the minimum example verification described in [Use FC Agent Sandbox with the SDK](../01.Getting%20Started/03.Quickstarts/01.Use%20FC%20Agent%20Sandbox%20with%20the%20SDK.md).
- Confirm that the SDK version meets the requirements in [Usage Constraints](../01.Getting%20Started/04.Usage%20Constraints.md).
- Confirm that the API you are using is listed as a compatible capability in [E2B Compatibility](../01.Getting%20Started/05.E2B%20Compatibility.md).
- Confirm that the API key, API URL, and domain are configured for the same region.
- Confirm that the template name, template ID, or sandbox ID is correct.

## Recommended information to provide

When troubleshooting sandbox creation, connection, or command execution issues, provide the following if possible:

- Alibaba Cloud account UID.
- Region.
- SDK language and version.
- Template name or template ID in use.
- Sandbox ID.
- Key error information and request time.
- A minimum reproducible code snippet.
- Whether you are using a custom template, an ACR EE image, NAS, OSS, or port access.

Do not paste full API keys, access tokens, database passwords, or user privacy data into tickets, screenshots, or logs.

## Support channels

You can get help through the Alibaba Cloud console, the Function Compute product support entry, the ticketing system, or customer groups. For contact details, see [Contact Us](05.Contact%20Us.md).
