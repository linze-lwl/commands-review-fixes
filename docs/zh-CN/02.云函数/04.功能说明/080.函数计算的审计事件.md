# 函数计算的审计事件

函数计算已与操作审计服务集成，您可以在操作审计中查询用户操作弹性伸缩产生的管控事件。函数计算支持将管控事件投递到日志服务SLS的LogStore或对象存储OSS的存储空间中，满足实时审计、问题回溯分析等需求。

操作审计记录了用户通过OpenAPI或控制台等方式操作云资源时产生的管控事件，函数计算支持在操作审计中查询的事件如下表所示。

| **事件名称** | **事件含义** |
| --- | --- |
| Create | 登录产品售卖页面进行的购买操作。 |
| CreateAlias | 创建别名。 |
| CreateCustomDomain | 创建自定义域名。 |
| CreateFunction | 创建函数。 |
| CreateService | 创建服务。 |
| CreateTrigger | 创建触发器。 |
| DeleteAlias | 删除别名。 |
| DeleteCustomDomain | 删除自定义域名。 |
| DeleteFunction | 删除函数。 |
| DeleteFunctionAsyncInvokeConfig | 删除一个服务下某个函数的异步调用配置。 |
| DeleteLayerVersion | 删除层版本。 |
| DeleteService | 删除服务。 |
| DeleteTrigger | 删除触发器。 |
| GetAlias | 查询别名信息。 |
| GetCustomDomain | 查询自定义域名。 |
| GetFunction | 查询函数信息。 |
| GetFunctionAsyncInvokeConfig | 查询一个服务下某个函数的异步调用配置。 |
| GetFunctionCode | 查询函数代码包。 |
| GetLayerVersion | 查询层版本信息。 |
| GetProvisionConfig | 查询预留配置。 |
| GetResourceTags | 查询资源所有的标签信息。 |
| GetService | 查询服务信息。 |
| GetTrigger | 查询触发器信息。 |
| ListAliases | 查询别名列表。 |
| ListFunctionAsyncInvokeConfigs | 查询一个服务下某个函数的所有异步配置。 |
| ListFunctions | 查询函数列表。 |
| ListLayers | 查询层列表。 |
| ListLayerVersions | 查询层的版本列表。 |
| ListProvisionConfigs | 查询预留配置列表。 |
| ListServices | 查询服务列表。 |
| ListTriggers | 查询触发器列表。 |
| Modify | 登录产品售卖页面进行的变配操作。 |
| OpenService | 开通服务。 |
| PutFunctionAsyncInvokeConfig | 创建或更新函数的异步调用配置。 |
| PutProvisionConfig | 设置预留配置。 |
| Renew | 实例或者资源包等的续费操作。 |
| TagResource | 给资源创建标签。 |
| UnTagResource | 删除资源的标签。 |
| UpdateAlias | 更新别名。 |
| UpdateCustomDomain | 更新自定义域名配置。 |
| UpdateFunction | 更新函数信息。 |
| UpdateService | 更新服务信息。 |
| UpdateTrigger | 更新触发器。 |
