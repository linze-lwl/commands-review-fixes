# 触发器管理

您可以在指定函数中创建触发器，使用触发器描述一组规则，当某个事件满足这些规则，事件源就会触发关联的函数。本文列举函数计算支持的所有触发器。

**

**说明**

单向集成触发器，例如[API网关触发器](https://help.aliyun.com/zh/functioncompute/fc/user-guide/api-gateway-trigger-overview)，只支持在事件源端配置，不支持在[函数计算控制台](https://fc.console.aliyun.com)修改。因此本文描述的触发器管理不包括单向集成触发器。

## 更新触发器配置

**

**说明**

已创建的[轻量消息队列（原 MNS）主题触发器](https://help.aliyun.com/zh/functioncompute/fc/user-guide/mns-topic-triggers)和[Tablestore触发器](https://help.aliyun.com/zh/functioncompute/fc/user-guide/tablestore-triggers)不支持使用以下方法修改。

1. 登录[函数计算控制台](https://fcnext.console.aliyun.com)，在左侧导航栏，选择**函数管理**>**函数列表**。
2. 在顶部菜单栏，选择地域，然后在**函数列表**页面，单击目标函数。
3. 在函数详情页面，选择**触发器**页签，然后在触发器列表单击目标触发器右侧**操作**列的**编辑**。
4. 在编辑触发器面板，根据需要修改触发器的配置，然后单击**确定**。

您也可以根据界面提示删除不再使用的触发器。
