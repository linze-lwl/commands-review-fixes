# 2017年功能发布记录

本文介绍函数计算2017年度发布的功能变更以及对应的文档动态。

## 2017年12月

| **功能名称** | **变更类型** | **功能描述** | **相关文档** |
| --- | --- | --- | --- |
| 与IoT产品的对接 | 新增 | 函数计算打通和IoT产品对接，通过IoT产品控制台可以配置函数计算作为后端处理，可以解决物联网、智能家居存在的问题：<br>- 数据不能即时处理<br>- 较重的技术架构<br>- 无法实时弹性扩容<br>- 高额成本支出<br>IoT和函数计算打通，函数计算作为后端处理，可以构建无服务器架构的应用场景，以及高弹性、低成本的场景。 | [物联网平台通信方式概述](https://help.aliyun.com/zh/iot/user-guide/overview-of-internet-of-things-platform-communication-mode#concept-2358399) |
| 全新版本控制台 | 新增 | 函数计算正式推出全新版本控制台，新版控制台提供计量数据和各种分类导航，整体界面紧凑简洁，编辑页面使用抽屉推出方式，减少大量界面交互跳转。让您在一个页面里操作多个功能。新推出的控制台的特性如下：<br>- 显示业务调用量<br>- 显示最新的触发器<br>- 显示最新模板详情<br>- 显示产品更新动态<br>- 抽屉式的页面<br>- 内容更丰富紧凑 | 无 |
| OSS控制台增加函数计算处理事件入口 | 优化 | OSS控制台新增函数计算处理事件入口，真正双向联通存储和计算，通过函数计算可以管理某个函数下的触发器事件，通过OSS的某Bucket可以管理该Bucket下的定义的事件，让两个产品真正融为一体，大大提升您的使用体验。 | [部署程序包](https://help.aliyun.com/zh/document_detail/161921.html#task-2474546) |
| 新增支持地域 | 优化 | 函数计算开通华南1地域，增加深圳节点。 | 无 |

## 2017年09月

| **功能名称** | **变更类型** | **功能描述** | **相关文档** |
| --- | --- | --- | --- |
| 集成表格存储 | 新增 | 函数计算打通了和表格存储访问，可以通过表格存储的表格增删改查等事件来触发函数计算的执行，真正完成智能数据的处理。 | [使用函数计算](https://help.aliyun.com/zh/tablestore/use-function-compute#concept-u5m-tqj-bfb) |
| 阿里云函数计算商业化 | 新增 | 函数计算是一个无服务器的全托管的运行环境，您只需编写代码并设置运行的条件，就可以弹性、安全地运行。函数计算会自行维护服务器资源、网络资源，以及消息分发和负载均衡等功能。其特点如下：<br>- 能大规模并行执行代码。<br>- 无需配置服务器、安装软件、部署容器。<br>- 容量自动配置、自动扩展机制，松散耦合。<br>- 事件触发运行代码方式。<br>- 按运行计费，不执行0费用。 | 无 |
| 华东1区开通 | 新增 | 阿里云函数计算开通华东1地域。 | 无 |
| 4大实用模板 | 新增 | 函数计算强力推出4大实用模板，例如打标签、爬虫算法、API网关对接返回串等多个模板，降低您的使用门槛，协助开发者更便捷地实现业务逻辑。 | 无 |
