# 我的客户端不关心函数执行结果，我不希望我的客户端一直等函数返回怎么办？

您可以使用函数计算的功能概览，[异步调用](https://help.aliyun.com/zh/functioncompute/fc/user-guide/asynchronous-invocation)会将您的请求加入到后端队列，客户端会立即返回。函数计算后端会将队列中的请求做并发调用。各SDK的async Invoke请参见：

- [Python SDK](https://github.com/aliyun/fc-python-sdk/blob/master/fc2/client.py#L626)
- [Node.js SDK](https://github.com/aliyun/fc-nodejs-sdk/blob/master/test/client.test.js#L304)
- [Java SDK](https://github.com/aliyun/fc-java-sdk/blob/master/src/test/java/com/aliyuncs/fc/FunctionComputeClientTest.java#L1373)
- [Golang SDK](https://github.com/aliyun/fc-go-sdk/blob/master/function_test.go#L20)
