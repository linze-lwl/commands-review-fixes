# 我在函数计算中访问数据库时，白名单如何配置？

访问RDS等其他类型的数据库时，如果需要配置IP白名单，请按照以下步骤操作。

- 如果您访问的是内网地址，需先开启函数计算服务的VPC访问能力，具体操作，请参见[配置网络](https://help.aliyun.com/zh/functioncompute/fc/user-guide/configure-network-settings)，然后找到目标数据库实例，将您为服务配置的交换机所在的IP地址段添加到白名单中。更多信息，请参见[设置白名单](https://help.aliyun.com/zh/rds/configure-a-whitelist)。
- 如果您访问的是公网地址，需先配置固定公网IP，具体操作，请参见[配置固定公网IP地址](https://help.aliyun.com/zh/functioncompute/fc/user-guide/configure-static-public-ip-addresses)，然后找到目标数据库实例，将此固定公网IP添加到白名单中。更多信息，请参见[设置白名单](https://help.aliyun.com/zh/rds/configure-a-whitelist)。
