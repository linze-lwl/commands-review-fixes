# HTTP请求如何进行JWT认证？

函数计算FC的HTTP触发器和自定义域名支持JWT认证，未通过认证的调用不收费。您也可以使用API网关的JWT插件进行认证。

相关文档详见：[HTTP触发器配置JWT认证](https://help.aliyun.com/zh/functioncompute/fc/user-guide/configure-jwt-authentication-for-an-http-trigger)，[自定义域名配置JWT认证](https://help.aliyun.com/zh/functioncompute/fc/user-guide/configure-jwt-authentication-for-custom-domain-names)，[使用API网关JWT认证插件](https://help.aliyun.com/zh/api-gateway/traditional-api-gateway/user-guide/jwt-authentication)。
