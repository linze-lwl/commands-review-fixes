# HTTP请求如何进行签名认证？

函数计算FC的HTTP触发器和自定义域名支持签名认证。您需要使用阿里云的AK和SK对HTTP请求进行签名，未通过认证的调用不收费。详情见[为HTTP触发器配置签名认证](https://help.aliyun.com/zh/functioncompute/fc/user-guide/configure-signature-authentication-for-http-triggers)和[为自定义域名配置签名认证](https://help.aliyun.com/zh/functioncompute/fc/user-guide/configure-signature-authentication-for-custom-domain-names)。
