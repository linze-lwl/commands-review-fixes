# 管理员界面功能详解

管理和运维人员通过管理后台（/admin/）对 开放平台进行全局管控。管理后台提供概览、权限管理（用户组管理、用户管理、用户空间管理）、设置（系统设置、API Key）和专题（养虾 FunClaw、养马 FunHermes）等功能模块。

## 管理后台概览

管理后台采用侧边栏导航。概览页面展示系统统计信息和最新动态，包含以下内容：

| 区块 | 说明 |
| --- | --- |
| 统计卡片 | 用户数量（含活跃用户）、用户空间（含已启用数）、用户组数量。 |
| 最新公告 | 展示系统公告和更新信息。 |
| 版本信息 | 当前版本号、发布日期、运行环境和更新内容。 |
| 技术支持 | 钉群支持、工单支持和快速帮助。 |
| 系统状态 | 系统服务、数据库和存储服务的运行状态。 |

## 权限管理

权限管理分为三个独立子页面，提供用户组、用户和用户空间的管理功能。

### 用户组管理

创建和管理用户组。支持搜索用户组名称或描述、创建用户组和刷新列表。

### 用户管理

集中管理所有用户的状态和权限。

| 功能 | 说明 |
| --- | --- |
| 用户列表 | 表格展示用户信息、所属用户组、资源权限、绑定沙箱、默认工作空间和状态。 |
| 搜索 | 按用户名称或邮箱搜索。 |
| 筛选 | 按用户组和状态（全部状态、已启用、已禁用）筛选。 |
| 批量导入 | 批量导入用户。 |
| 新建用户 | 创建新用户并分配用户组。 |

### 用户空间管理

创建和管理用户空间。支持搜索用户空间名称或描述、按状态筛选（已启用/已禁用）和新建用户空间。

## 系统设置

系统设置页面包含三个配置区块：

### 管理员配置

| 配置项 | 说明 |
| --- | --- |
| 管理员用户名 | 管理后台的登录用户名（默认 admin，只读）。 |
| 管理员密码 | 支持修改管理后台登录密码，密码长度至少 8 位，包含字母、数字和特殊字符。 |
| 最后登录时间 | 展示管理员最近一次登录的时间。 |

### 阿里云 AK/SK 配置

配置平台访问阿里云 API 服务所需的 AccessKey。

| 配置项 | 说明 |
| --- | --- |
| 阿里云账号 UID | 与当前凭证所属账号一致，保存访问密钥后自动更新。 |
| AccessKey ID | 阿里云 AccessKey ID。 |
| AccessKey Secret | 阿里云 AccessKey Secret。 |

**

**重要**

首次使用管理后台时，系统会强制跳转到系统设置页面要求配置 AK/SK。未配置 AK/SK 前，无法访问管理后台的其他页面。

### 系统认证配置

通过开关控制三种认证方式：

| 认证方式 | 说明 |
| --- | --- |
| 用户注册登录 | 允许用户通过邮箱或手机号注册账号（默认开启）。 |
| SSO 单点登录 | 企业员工通过 SSO 统一身份认证登录。单击**编辑 SSO 配置**进行协议对接配置。 |
| 企业微信登录 | 允许用户通过企业微信扫码或授权登录。单击**编辑企业微信配置**进行配置。 |

## API Key 管理

创建和管理 API Key，通过 API 以编程方式访问管理后台功能。

### API Key 管理

每个管理员最多创建 10 个 API Key。

| 功能 | 说明 |
| --- | --- |
| 创建 API Key | 创建新的 API Key。 |
| 脱敏展示 | 列表中仅展示 Key 前缀，避免泄露。 |
| 启用或禁用 | 随时开关某条 Key。 |
| 删除 | 永久删除 Key。 |

### API 文档与示例

提供 Swagger API 文档链接和 cURL 示例。所有管理员 API 通过 Authorization: Bearer 头部传递 API Key 认证。

**

**警告**

API Key 拥有管理员的完整权限，请妥善保管，不要在前端代码或公开仓库中暴露。

## 专题

管理后台的「专题」分区提供以下产品线入口：

- **养虾 FunClaw**— 面向 OpenClaw 的专题配置管理，详见[专题-FunClaw](https://help.aliyun.com/zh/functioncompute/fc/special-feature-funclaw)。
-

每个专题入口在侧边栏展开独立子菜单，包含模型配置、存储配置、技能配置等详细配置页面。
