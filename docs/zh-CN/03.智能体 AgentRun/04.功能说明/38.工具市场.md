# 工具市场

工具市场提供预置的 Skill、MCP 工具和 Function Call 工具，覆盖数据处理、安全防护、存储管理等场景，支持一键安装部署，无需逐一创建和部署，即可快速为 Agent 扩展能力。

## 支持的工具类型

工具市场提供以下三种类型的工具，根据 Agent 使用的大模型能力选择适配类型：

| 工具类型 | 说明 | 适用场景 |
| --- | --- | --- |
| Skill | 技能包形式的工具，将复杂业务逻辑封装为可复用的技能包，在沙箱环境中安全执行。AgentRun 特有的工具类型。 | 需要动态挂载和沙箱隔离运行的场景 |
| MCP 工具 | 基于 MCP（Model Context Protocol）协议的工具，为大模型与外部工具、数据源之间提供标准化通信。 | 支持 MCP 协议的大模型平台 |
| Function Call 工具 | 基于函数调用能力的工具，通过 OpenAPI 接口直接调用。 | 支持 Function Call 的大模型 |

## 前提条件

- 已开通函数计算 FC 服务。AgentRun 基于函数计算 FC 运行，使用前需确保已开通该服务。
- 已登录[AgentRun 控制台](https://functionai.console.aliyun.com/cn-hangzhou/agent/runtime/agent-list)。
- 具备工具管理权限。如需了解权限配置，请联系账号管理员。

## 访问工具市场

1. 登录[AgentRun 控制台](https://agentrun.console.aliyun.com)。
2. 在左侧导航栏，选择**工具与Skills**。
3. 单击**工具市场**标签页。

## 浏览和筛选工具

工具市场提供以下方式快速找到所需工具：

- **搜索**：在搜索框中输入工具名称或关键词，定位目标工具。
- **按类型筛选**：通过**全部类型**下拉框筛选工具，支持以下选项：
  
  - **全部类型**：显示所有类型的工具。
  - **Skill**：技能包形式的工具。
  - **Function Call**：基于函数调用能力的工具。
  - **MCP协议**：基于 Model Context Protocol 协议的工具。

**工具市场**中的工具卡片展示工具名称、作者、工具类型、收藏数和功能描述。单击工具卡片可查看详情。

**我的工具**中的工具卡片展示工具名称、类型（或来源和类型）、描述和操作按钮。

## 安装工具

1. 在工具市场中，单击目标工具卡片查看详情。
2. 确认工具信息后，单击**安装**。
3. 在弹出的工具安装对话框中，填写以下信息：
  
  | 参数 | 是否必填 | 说明 |
  | --- | --- | --- |
  | 工具名称 | 是 | 系统自动生成默认名称（如`fcai-start-mcp-python-hello-world-3r1h`），可修改为更易识别的名称。名称必须以字母或下划线开头，可包含字母、数字、下划线和连字符。 |
  | 描述 | 否 | 填写工具用途或备注，便于团队协作时识别。 |
4. 确认工具名称和描述后，单击**确认安装**。
5. 安装完成后，工具将出现在**我的工具**标签页中。

**

**说明**

- 安装后的工具可以直接在创建或编辑 Agent 时添加使用。
- 如果工具依赖第三方服务（如高德地图 API），安装完成后在工具详情页的**概览**标签页中，通过**高级配置**>**凭证配置**设置访问凭据。

### 常见问题

| 现象 | 可能原因 | 排查步骤 |
| --- | --- | --- |
| 安装失败，提示部署错误 | 函数计算 FC 服务未开通，或当前地域不支持该工具 | 1. 确认已开通函数计算 FC 服务。<br>2. 检查工具支持的地域列表，切换至支持的地域后重试。 |
| 安装成功但调用失败，返回认证错误 | 访问凭据配置错误或 API Key 已过期 | 1. 在**我的工具**中找到目标工具，单击进入详情页。<br>2. 在**概览**标签页中，通过**高级配置**>**凭证配置**检查凭据配置，确认 API Key 正确且未过期。<br>3. 如需更换凭据，单击编辑，重新配置访问凭据后保存。 |
| 安装后工具未出现在我的工具中 | 安装过程可能未完成，或浏览器缓存未刷新 | 1. 刷新页面后重新检查**我的工具**标签页。<br>2. 如仍未显示，返回工具市场重新安装。 |

## 查看工具详情

安装成功后，在**我的工具**标签页中，单击目标工具卡片进入工具详情页。详情页包含以下标签页：

| 标签页 | 内容 |
| --- | --- |
| 概览 | 显示工具基本信息（名称、ID、工作空间、描述、连接类型）、工具状态、网络配置、启动配置、执行环境、资源配置、高级配置（含凭证配置）。支持编辑配置。 |
| 可观测 | 运行监控面板，支持查看总调用量、平均/最大响应时间、错误数、调用趋势图，以及 CPU、内存、网络流量等资源使用情况。 |
| 使用案例 | 展示工具的集成指南和调用方式。具体内容因工具类型而异，详见下方说明。 |

### 使用案例

在工具详情页中，单击**使用案例**标签页查看工具的集成指南。不同类型的工具展示不同内容：

- **MCP 工具**：提供多平台集成指南（百炼、Langstudio、Cursor、Cline、Modelscope、Dify），展示各平台的配置步骤和 JSON 配置。
- **Function Call 工具**：提供 OpenAPI 接口说明及调用示例，包含接口地址和请求参数信息。
