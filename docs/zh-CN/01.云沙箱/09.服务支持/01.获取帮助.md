# 获取帮助

如果在使用 云沙箱时遇到问题，建议先完成最小示例验证，并收集必要排查信息，再通过阿里云控制台、工单或客户群获取支持。

## 提交问题前先确认

- 是否已按[通过 SDK 使用云沙箱](../01.开始使用/03.快速入门/01.通过%20SDK%20使用云沙箱.md)完成最小示例验证。
- SDK 版本是否满足[使用约束](../01.开始使用/04.使用约束.md)。
- 使用的接口是否在[E2B 兼容说明](../01.开始使用/05.E2B%20兼容说明.md)中列为兼容能力。
- API Key、API URL、域名是否配置为同一地域。
- 模板名称、模板 ID 或 Sandbox ID 是否正确。

## 建议提供的信息

排查 Sandbox 创建、连接或命令执行问题时，建议提供：

- 阿里云账号 UID。
- 地域。
- SDK 语言和版本。
- 使用的模板名称或模板 ID。
- Sandbox ID。
- 关键错误信息和请求时间。
- 最小可复现代码片段。
- 是否使用自定义模板、ACR EE 镜像、NAS、OSS 或端口访问。

不要在工单、截图或日志中粘贴完整 API Key、Access Token、数据库密码或用户隐私数据。

## 支持渠道

可通过阿里云控制台、函数计算产品支持入口、工单系统或客户群获取帮助。联系方式参见[联系我们](05.联系我们.md)。
