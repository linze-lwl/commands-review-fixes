# 构建模板

构建模板会把基础镜像或模板定义提交到云沙箱控制面，生成后续创建沙箱可复用的模板。

## 构建并打印模板 ID

```python
import os
import time

from e2b import Template, default_build_logger

build = Template.build(
    Template().from_image(os.environ["FROM_IMAGE"]),
    name=f"agent-runtime-{int(time.time())}",
    cpu_count=2,
    memory_mb=2048,
    skip_cache=False,
    on_build_logs=default_build_logger(),
    api_key=os.environ["E2B_API_KEY"],
    api_url=os.environ["E2B_API_URL"],
    domain=os.environ["E2B_DOMAIN"],
)

print("template_id:", build.template_id)
print("build_id:", build.build_id)
```

## 构建后验证

```python
from e2b import Sandbox

sandbox = Sandbox.create(
    template=build.template_id,
    api_key=os.environ["E2B_API_KEY"],
    api_url=os.environ["E2B_API_URL"],
    domain=os.environ["E2B_DOMAIN"],
)

try:
    result = sandbox.commands.run("python3 --version")
    print(result.stdout)
finally:
    sandbox.kill()
```

## 使用建议

构建完成后应立即创建一个沙箱做冒烟验证。不要只根据构建成功判断模板可用于生产任务。
