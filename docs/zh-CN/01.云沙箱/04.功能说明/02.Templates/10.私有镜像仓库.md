# 私有镜像仓库

云沙箱支持从满足网络要求的私有镜像仓库构建模板。当前官方文档推荐使用与云沙箱同账号、同地域、可通过 VPC 访问的 ACR EE 镜像。

## 网络要求

- ACR EE 实例已绑定专有网络。
- 镜像仓库、VPC、vSwitch 与云沙箱位于同一地域。
- VPC 和安全组规则允许云沙箱访问镜像仓库。
- 镜像地址使用 VPC 内网访问域名。

## 示例

```bash
export FROM_IMAGE="<acr-ee-vpc-domain>/<namespace>/<image>:<tag>"
```

```python
build = Template.build(
    Template().from_image(os.environ["FROM_IMAGE"]),
    name="agent-runtime",
    api_key=os.environ["E2B_API_KEY"],
    api_url=os.environ["E2B_API_URL"],
    domain=os.environ["E2B_DOMAIN"],
)
```

## 使用建议

构建失败时，优先检查镜像地址、地域、VPC 绑定、安全组和仓库访问权限。不要在文档或代码中硬编码真实镜像仓库地址。
