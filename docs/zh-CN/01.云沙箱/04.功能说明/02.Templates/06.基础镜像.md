# 基础镜像

基础镜像是自定义模板的起点。云沙箱当前推荐使用可通过 VPC 访问的 ACR EE 镜像作为构建来源。

## 镜像地址格式

```text
<acr-ee-vpc-domain>/<namespace>/<image>:<tag>
```

## 构建自定义模板

```python
import os
import time

from e2b import Template

connection = {
    "api_key": os.environ["E2B_API_KEY"],
    "api_url": os.environ["E2B_API_URL"],
    "domain": os.environ["E2B_DOMAIN"],
}

build = Template.build(
    Template().from_image(os.environ["FROM_IMAGE"]),
    name=f"my-template-{int(time.time())}",
    **connection,
)

print(build.template_id)
```

TypeScript 示例：

```typescript
import { Template } from "e2b";

const build = await Template.build(
  Template().fromImage(process.env.FROM_IMAGE),
  `my-template-${Date.now()}`,
  {
    apiKey: process.env.E2B_API_KEY,
    apiUrl: process.env.E2B_API_URL,
    domain: process.env.E2B_DOMAIN,
  },
);

console.log(build.templateId);
```

## 使用建议

基础镜像应尽量稳定、可追溯。生产环境建议使用明确版本号或 digest，避免使用会漂移的 `latest`。
