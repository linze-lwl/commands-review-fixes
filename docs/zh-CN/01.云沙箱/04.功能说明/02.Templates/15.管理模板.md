# 管理模板

模板管理包括查看模板、创建模板、跟踪构建状态，以及使用标签和别名组织模板版本。

## 查看模板列表

使用 E2B CLI：

```bash
e2b template list
```

也可以通过 E2B SDK 查询模板。具体接口边界参见[E2B 兼容说明](../../01.开始使用/05.E2B%20兼容说明.md)。

## 创建模板

使用 `Template.build` 创建自定义模板。详见[自定义模板](03.自定义模板.md)。

## 构建状态和日志

构建模板时，应保留构建 ID 和模板 ID，便于排查构建失败、镜像拉取失败或依赖安装失败等问题。

## 标签和别名

标签和别名适合区分模板版本、业务场景和灰度阶段。生产环境建议先创建新模板并验证，再切换业务使用的模板别名。
