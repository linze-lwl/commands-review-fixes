# 构建缓存

构建缓存用于减少重复模板构建时间。依赖稳定、基础镜像不变时，可以复用缓存；需要强制重建时，可以关闭缓存。

## 使用缓存构建

```python
build = Template.build(
    Template().from_image(os.environ["FROM_IMAGE"]),
    name="agent-runtime",
    skip_cache=False,
    api_key=os.environ["E2B_API_KEY"],
    api_url=os.environ["E2B_API_URL"],
    domain=os.environ["E2B_DOMAIN"],
)
```

## 强制重新构建

```python
build = Template.build(
    Template().from_image(os.environ["FROM_IMAGE"]),
    name="agent-runtime",
    skip_cache=True,
    api_key=os.environ["E2B_API_KEY"],
    api_url=os.environ["E2B_API_URL"],
    domain=os.environ["E2B_DOMAIN"],
)
```

## 使用建议

开发调试阶段可以按需关闭缓存，确保依赖变更被重新构建。生产模板建议使用可追溯镜像版本，并在构建完成后做沙箱冒烟验证。
