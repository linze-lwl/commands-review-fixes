# 模板名称

模板名称用于在创建沙箱时引用运行环境。好的命名可以降低模板误用和版本混乱。

## 命名建议

建议包含业务场景、语言或基础镜像、关键依赖版本和日期：

```text
agent-python313-20260704
code-review-node22-20260704
data-analysis-py313-v1
```

## 创建沙箱时使用模板名称

```typescript
const sandbox = await Sandbox.create("agent-python313-20260704", {
  apiKey: process.env.E2B_API_KEY,
  apiUrl: process.env.E2B_API_URL,
  domain: process.env.E2B_DOMAIN,
});
```

## 使用建议

生产环境不要覆盖正在使用的模板。推荐创建新模板、完成验证、再切换业务引用。
