# 列出沙箱

列出沙箱用于查看当前账号下的沙箱实例。默认情况下，CLI 返回运行中的沙箱。

```bash
e2b sandbox list
```

如需同时查看运行中和已暂停的沙箱，请指定状态：

```bash
e2b sandbox list --state running,paused
```

可以按元数据过滤：

```bash
e2b sandbox list --metadata key1=value1,key2=value2
```

可以限制返回数量：

```bash
e2b sandbox list --limit 10
```

可以切换输出格式：

```bash
e2b sandbox list --format pretty
e2b sandbox list --format json
```

该命令适合做本地调试和资源排查。生产业务系统不应依赖 CLI 管理沙箱生命周期，应使用 SDK 或服务端控制逻辑。
