# 创建沙箱

使用 CLI 可以从指定模板创建云沙箱，并连接到交互式终端，适合本地验证模板、命令和网络访问。

```bash
e2b sandbox create <template>
```

例如，使用 `base` 模板创建沙箱：

```bash
e2b sandbox create base
```

该命令会：

1. 从指定模板创建一个新沙箱。
2. 将本地终端连接到该沙箱。
3. 在终端连接期间保持沙箱运行。
4. 退出终端后自动终止该沙箱。

如果需要只创建沙箱并由业务代码管理生命周期，请使用 SDK 的 `Sandbox.create()`。不同 CLI 版本的参数可能存在差异，执行前可查看帮助：

```bash
e2b sandbox create --help
```
