# 「不兼容」图表与可视化

E2B Code Interpreter 支持在执行结果中返回图表和富媒体结果。云沙箱当前未确认兼容该能力，不应在官方功能说明中作为正式能力引导开发者使用。

需要可视化时，可以先在沙箱内生成图片文件，再通过 Filesystem 或下载 URL 取回：

```python
sandbox.run_code("""
import matplotlib.pyplot as plt

plt.plot([1, 2, 3], [1, 4, 9])
plt.savefig('/tmp/chart.png')
""")
```

随后使用[上传和下载文件](../04.Filesystem/03.上传和下载文件.md)下载 `/tmp/chart.png`。
