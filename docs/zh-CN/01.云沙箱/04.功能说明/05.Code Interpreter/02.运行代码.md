# 运行代码

Code Interpreter 用于在 Sandbox 中运行代码，适合数据分析、代码解释器和 Agent 代码执行场景。

## 创建 Sandbox

可以使用 `code-interpreter-v1` 模板创建 Sandbox：

```python
import os

from e2b_code_interpreter import Sandbox

sandbox = Sandbox.create(
    template="code-interpreter-v1",
    api_key=os.environ["E2B_API_KEY"],
    api_url=os.environ["E2B_API_URL"],
    domain=os.environ["E2B_DOMAIN"],
)
```

## 执行代码

```python
execution = sandbox.run_code("print('hello')")
stdout = "".join(execution.logs.stdout or [])
print(stdout.strip())
```

## 上下文管理

执行上下文可用于在多次代码执行之间保留变量和状态。适合多轮数据分析和交互式代码解释器。

## 输出行为

- 支持 stdout/stderr timestamp。
- 支持 execution count。
- 裸表达式进入 `execution.text`，`print` 进入 stdout。
- 支持多次 `run_code` 之间变量保持。
