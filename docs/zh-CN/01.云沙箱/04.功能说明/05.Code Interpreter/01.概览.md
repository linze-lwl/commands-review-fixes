# 概览

Code Interpreter 用于在沙箱中执行代码并获取结构化结果，适合数据分析、代码解释器、Notebook 类体验和 Agent 生成代码后的即时运行。

## SDK 入口

Code Interpreter 使用独立 SDK 入口：

```typescript
import { Sandbox } from "@e2b/code-interpreter";
```

Python 使用：

```python
from e2b_code_interpreter import Sandbox
```

连接参数仍然使用 `E2B_API_KEY`、`E2B_API_URL` 和 `E2B_DOMAIN`。

## 最小示例

```python
import os

from e2b_code_interpreter import Sandbox

sandbox = Sandbox.create(
    api_key=os.environ["E2B_API_KEY"],
    api_url=os.environ["E2B_API_URL"],
    domain=os.environ["E2B_DOMAIN"],
)

try:
    execution = sandbox.run_code("print('hello')")
    print("".join(execution.logs.stdout or []).strip())
finally:
    sandbox.kill()
```
