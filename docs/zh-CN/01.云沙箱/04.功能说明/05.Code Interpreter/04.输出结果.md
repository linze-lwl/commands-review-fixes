# 输出结果

Code Interpreter 的执行结果包含 stdout、stderr、表达式结果和错误信息。业务侧应同时处理成功输出和失败信息。

## 读取 stdout

```python
execution = sandbox.run_code("print('hello')")
stdout = "".join(execution.logs.stdout or [])

print(stdout.strip())
```

## 读取表达式结果

```python
execution = sandbox.run_code("1 + 1")
print(execution.text)
```

## 处理错误

```python
execution = sandbox.run_code("raise ValueError('bad input')")

if execution.error:
    print(execution.error.name)
    print(execution.error.value)
```

## 使用建议

面向用户展示结果时，应区分 stdout、表达式结果和错误。对 Agent 场景，建议把 `execution.error` 作为重试或修正代码的输入，而不是只返回原始报错。
