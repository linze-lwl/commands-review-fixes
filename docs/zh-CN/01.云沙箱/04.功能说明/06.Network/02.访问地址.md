# 访问地址

部分示例会输出 Sandbox 相关地址，例如 `sandbox_domain` 和 `envd_api_url`。这些字段主要用于调试和确认 Sandbox 创建结果。

## 示例

自定义模板验证脚本中包含以下输出：

```python
print(f"sandbox_id: {sandbox.sandbox_id}")
print(f"sandbox_domain: {sandbox.sandbox_domain}")
print(f"envd_api_url: {sandbox.envd_api_url}")
```

TypeScript 示例：

```typescript
console.log(`sandbox_id: ${sandbox.sandboxId}`);
console.log(`sandbox_domain: ${sandbox.sandboxDomain}`);
```

## 使用建议

- 业务访问 Sandbox 端口时，优先使用 `sandbox.getHost(port)`。
- 完整端口访问示例参见[公网访问](03.公网访问.md)。
- 不建议依赖调试字段拼接内部地址，具体字段稳定性请参见 SDK 和兼容说明。
