# 终止命令

终止命令用于停止沙箱中正在运行的后台进程或 PTY 会话。它不会终止整个沙箱。

## 终止后台命令

```typescript
const handle = await sandbox.commands.run("sleep 600", {
  background: true,
});

const killed = await sandbox.commands.kill(handle.pid);
console.log(killed);
```

## 清理所有后台命令

```typescript
const processes = await sandbox.commands.list();

for (const process of processes) {
  await sandbox.commands.kill(process.pid);
}
```

## 使用建议

长任务应设置业务侧超时，并在超时后主动终止命令。终止命令只释放进程资源；如果整个任务结束，还应调用 `sandbox.kill()` 释放沙箱。
