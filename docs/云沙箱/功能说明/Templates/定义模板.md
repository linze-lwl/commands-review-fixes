# 定义模板

定义模板用于描述构建沙箱运行环境的步骤。云沙箱当前可通过 E2B SDK 的 `Template.build()` 提交模板构建。

## 从镜像定义模板

```python
from e2b import Template

template = Template().from_image("<acr-ee-vpc-domain>/<namespace>/<image>:<tag>")
```

## 提交构建

```python
import os

from e2b import Template

build = Template.build(
    template,
    name="agent-runtime",
    cpu_count=2,
    memory_mb=2048,
    api_key=os.environ["E2B_API_KEY"],
    api_url=os.environ["E2B_API_URL"],
    domain=os.environ["E2B_DOMAIN"],
)
```

## 使用建议

模板定义应只包含运行环境内容。任务输入、用户数据和临时状态应在创建沙箱后写入，避免模板和业务数据耦合。
